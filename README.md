# Getting Started with Docker Compose

Follow the steps below to start the project locally.

---

## Requirements
- Docker installed
- (Optional but recommended) Docker Desktop
- A MySQL/PostgreSQL database client (e.g. **DBeaver**)

---

## Start the project

Open a terminal and navigate to the project directory. Then start the containers with:

```bash
docker compose up
```

## Enjoy

- http://localhost:6333/dashboard (qdrant)
